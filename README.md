# MiMo Doc QA Agent Benchmark

本项目用于验证 MiMo 系列模型在中文文档理解与 Agent 工作流中的表现，重点关注长文本阅读、知识库问答、摘要生成、结构化信息提取和中文指令遵循能力。

## 当前进展

已完成第一轮轻量级任务设计与初步验证，测试内容包括：

- 中文产品需求文档理解
- 文档摘要生成
- 关键信息提取
- 基于文档的问答
- 简单 Doc QA Demo
- 评测维度设计

## 第一轮测试任务

任务：基于一份中文产品需求文档，完成摘要生成、关键信息提取和问答。

核心测试点：

- 是否能理解中文业务文档
- 是否能提取目标用户、核心功能、约束条件
- 是否能基于文档回答问题，避免脱离上下文
- 是否能输出结构化结果
- 是否适合嵌入 Agent 工作流

## Demo

第一轮可运行 Demo 位于：`demo/simple-doc-qa/`

运行方式：

1. 下载或克隆仓库；
2. 打开 `demo/simple-doc-qa/index.html`；
3. 粘贴中文文档内容；
4. 输入问题，测试简单关键词匹配式文档问答流程。

该 Demo 用于模拟 AI Agent 在读取中文文档后进行问答和摘要的基础流程。
