# Evaluation

## Round 1: 中文文档问答 Agent

| 维度 | 观察结果 | 评分 |
|---|---|---|
| 中文文档理解 | 能识别项目背景、目标用户和核心功能 | 4/5 |
| 摘要生成 | 能压缩为简洁摘要，保留主要信息 | 4/5 |
| 信息提取 | 能提取目标用户、核心功能和约束条件 | 4/5 |
| 问答相关性 | 回答基本围绕文档内容展开 | 4/5 |
| 防幻觉能力 | 通过提示词约束，可减少编造信息 | 3/5 |
| 结构化输出 | 可输出 JSON 或表格格式 | 4/5 |
| Token 消耗 | 长文档和多轮问答场景下消耗较高 | TBD |

## 初步结论

第一轮测试说明，MiMo 类中文模型适合用于中文文档理解、摘要生成、知识库问答和结构化信息提取等 Agent 场景。

后续计划继续测试更长文档、多文档检索、多轮问答和复杂业务规则分析。
