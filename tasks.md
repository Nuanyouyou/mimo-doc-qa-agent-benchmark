# Test Tasks

## Task 1: 中文产品需求文档问答

使用中文产品需求文档，测试模型是否能完成摘要、信息提取和基于文档的问答。

### 输入材料

一份模拟产品需求文档，包含：

1. 产品背景
2. 目标用户
3. 核心功能
4. 使用场景
5. 约束条件
6. 后续计划

### 任务要求

请 AI Agent 完成以下任务：

1. 用 150 字以内总结文档；
2. 提取目标用户；
3. 提取核心功能列表；
4. 提取潜在风险；
5. 根据文档回答 5 个问题；
6. 输出结构化 JSON；
7. 生成一份简短评测记录。

### 评测点

- 是否准确理解中文文档
- 是否能基于原文回答
- 是否避免编造不存在的信息
- 是否能输出结构化内容
- 是否适合用于企业文档分析 Agent
